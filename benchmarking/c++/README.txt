Installation instructions:

Step 1: Clone gurls repository locally (https://github.com/LCSL/GURLS)
Step 2: Copy this directory's contents into GURLS/gurls++/demo/
Step 3: Follow the GURLS++ installation instructions
Step 4: In the runCXX.sh file, change the gurlsData variable to wherever you have the julia GURLS repository cloned
Step 5: In runCXX.sh, uncomment the appropriate dataset you'd like to benchmark.
Step 6: Run runCXX.sh! All data is printed to STDOUT.
